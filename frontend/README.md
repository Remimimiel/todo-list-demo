## Run

1. Download denpendency

    ```sh {.line-numbers}
    # cnpm install
    npm install
    ```

2. Create `src/api/host.js` and specify the host info

3. Get start

    ```sh {.line-numbers}
    yarn start
    ```

4. Open browser and visit `http://localhost:3000`

## Deploy

1. Download denpendency

    ```sh {.line-numbers}
    cnpm install
    ```

2. Create `src/api/host.js` and specify the host info

3. Build

    ```sh {.line-numbers}
    yarn build
    ```

4. Get start

    ```sh {.line-numbers}
    # yarn add serve
    serve build -s -l 80
    ```

5. Open browser and visit `http://localhost`
